Reactjs nodejs upload image — How to upload image using reactjs and nodejs (multer)

Mahesh Joshi http://maheshjoshi.me/
Jul 30, 2018 · 2 min read

I am writing this article just because i couldn’t find well documented article after googling.

There are two sides. Client and Server. Reactjs is a client side while nodejs is a server side.

So, the bottom line is we are going to upload an image from reactjs and catch it in nodejs.

That’s it.

Reactjs Code :

import React from 'react'
const axios = require("axios");

class ReactUploadImage extends React.Component {

    constructor(props) {
        super(props);
        this.state ={
            file: null
        };
        this.onFormSubmit = this.onFormSubmit.bind(this);
        this.onChange = this.onChange.bind(this);
    }
    onFormSubmit(e){
        e.preventDefault();
        const formData = new FormData();
        formData.append('myImage',this.state.file);
        const config = {
            headers: {
                'content-type': 'multipart/form-data'
            }
        };
        axios.post("/upload",formData,config)
            .then((response) => {
                alert("The file is successfully uploaded");
            }).catch((error) => {
        });
    }
    onChange(e) {
        this.setState({file:e.target.files[0]});
    }

    render() {
        return (
            <form onSubmit={this.onFormSubmit}>
                <h1>File Upload</h1>
                <input type="file" name="myImage" onChange= {this.onChange} />
                <button type="submit">Upload</button>
            </form>
        )
    }
}

export default ReactUploadImage
